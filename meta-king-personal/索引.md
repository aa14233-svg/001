## 数据流

```
用户任务
  │
  ▼
Planner(听证三步) ──→ 任务 DAG
  │
  ▼
Thinker(probe) ──→ 需求-环境对账表 + 缺口清单
  │
  ▼
ThinkerRepair(自修闸门 Gate A/B/C)
  │
  ├─ 修复成功 ──→ Builder(执行)
  │
  └─ 修复失败 ──→ Searcher(检索) ──→ Builder(执行)
                                              │
                                              ▼
                                          Auditor(四维审查)
                                              │
                                              ▼
                                          Recorder(task_id 持久化)
                                              │
                                              ▼
                                          PatternBank(候选升级判定)
```
