# King-Skill 技能群系系统

> 以 King Agent 为中心拓扑展开的大智能体技能路由群系系统。

## 一、系统概述

King-Skill 是一套以 **King Agent（一体超强智能体）** 为中心拓扑，整合 **cc-switch、token-stats、master-engineer-hub** 三大核心技能，辅以 8 大技能组、20+ Agent 集群、LSO-DAO 五境递进基础设施的完整 **AI 智能体技能路由生态**。

### 核心哲学

> **不持一物，而万物皆备。不预一器，而万器待召。**

### 技术栈

| 组件 | 技术 |
|------|------|
| 路由引擎 | master-engineer-hub（假借修真 + 机械降神） |
| Provider 管理 | cc-switch（10 MCP 工具） |
| Token 审计 | token-stats（6 MCP 工具） |
| 基础设施 | LSO-DAO 五境递进 + 呼吸管道 |
| 执行层 | 20+ Agent 集群（7 核心 + 8 工程 + 5 三合一） |
| 技能层 | SkillHub 8 大技能组（20+ SKILL 文件） |
| 质量体系 | Hook+Loop + D1/D2/D3 三级验收 |
| 经济体系 | 引力盆 0 token 直通 90% 请求 |
