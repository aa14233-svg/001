# LSO-DAO 五境递进架构

> LSO-DAO 是 King Agent 的"修炼体系"——任务不是一次性完成，而是分五境递进，逐层深入。

## 一、五境递进总览

```
筑基 → 金丹 → 元婴 → 化神 → 渡劫
  │       │       │       │       │
  类型    安全    门控+   技能    内省+
  推导    扫描    域映射  路由    演化
                     │               验收
                    负载均衡
```

## 二、五境详解

### 第一境：筑基（类型推导）

**目标**：确定任务类型，建立基础认知

```
输入任务
  │
  ├── 任务类型识别
  │    ├── 代码任务
  │    ├── 文档任务
  │    ├── 设计任务
  │    ├── 研究任务
  │    └── 管理任务
  │
  ├── 上下文提取
  │    ├── 项目结构
  │    ├── 依赖关系
  │    └── 历史记录
  │
  └── 基础计划生成
```

### 第二境：金丹（安全扫描）

**目标**：安全检查，确保任务在安全边界内执行

```
安全扫描
  │
  ├── 路径检查
  │    ├── 路径禁区防护
  │    ├── 系统文件保护
  │    └── 配置只读保护
  │
  ├── 高危操作拦截
  │    ├── 危险命令识别
  │    ├── 网络操作审核
  │    └── 数据删除确认
  │
  └── 权限验证
```

### 第三境：元婴（五行门控 + 域映射）

**目标**：门控决策 + 领域映射，确定执行路径

```
五行门控
  │
  ├── 金（代码）→ 编码/开发域
  ├── 木（文档）→ 文档/写作域
  ├── 水（数据）→ 数据/分析域
  ├── 火（设计）→ 设计/创意域
  └── 土（部署）→ 部署/运维域

域映射
  └── 门控结果 → 映射到 8 大技能域
       └── 激活对应技能组
```

### 第四境：化神（技能路由 + 负载均衡）

**目标**：路由到具体技能 + 均衡负载

```
技能路由
  │
  ├── 在激活域中匹配技能
  │    ├── master-engineer-hub 路由
  │    ├── cc-switch Provider 路由
  │    └── token-stats Token 预算检查
  │
  └── 负载均衡
       ├── 当前 Provider 负载检查
       ├── Token 预算剩余检查
       └── 最优 Agent 选择
```

### 第五境：渡劫（内省 + 演化 + 验收）

**目标**：自我验证 + 经验积累 + 质量验收

```
内省（反射）
  │
  ├── 执行过程检查
  │    ├── 是否偏离计划
  │    ├── 是否有错误
  │    └── 是否需要调整
  │
  ├── 演化（学习）
  │    ├── 记录成功模式 → 引力盆
  │    ├── 记录失败模式 → 避免重复
  │    └── 更新技能路由权重
  │
  └── 验收（D1/D2/D3）
       ├── D1：阶段验收
       ├── D2：模块验收
       └── D3：总体验收
```

## 三、五境与 System Prompt 的对应关系

| 五境 | 对应 System Prompt 层 | 加载时机 |
|------|---------------------|---------|
| 筑基 | 基础 System Prompt | 任务初始化 |
| 金丹 | 安全规则（core-rules） | 安全扫描 |
| 元婴 | 增强版 System Prompt | 门控决策 |
| 化神 | SkillHub + SKILL.md | 技能路由 |
| 渡劫 | 验收规则 + 引力盆 | 完成时 |

## 四、呼吸管道

五境递进运行在"呼吸管道"之上，确保任务可暂停、可恢复、可清理：

```
吸入（吸）→ 任务到达，准备资源
  │
储存（存）→ 状态缓存，上下文持久化
  │
呼出（呼）→ 执行任务，产出结果
  │
化合（化）→ 结果整合，数据关联
  │
归墟（归）→ 资源释放，临时数据清理
```

## 五、Agent 集群在五境中的分布

| 五境 | 参与的 Agent | 负责 |
|------|-------------|------|
| 筑基 | 全局 Agent | 类型识别，计划生成 |
| 金丹 | 安全 Agent | 安全扫描，权限验证 |
| 元婴 | 门控 Agent | 域映射，决策路由 |
| 化神 | 领域 Agent（15+） | 技能执行，任务处理 |
| 渡劫 | 验收 Agent | 检查，审计，学习 |
